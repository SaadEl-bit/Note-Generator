2. Formules et codes de réponse à mémoriser
O = D + E + N + H
Objectif complet : dimension, échelle, norme, horizon.
PODC = Planifier → Organiser → Diriger
→ Contrôler
Code de classement des fonctions de gestion.
PESTEL = P + E + S + T + E + L
Code de classement du macro-environnement.
SWOT : Interne = Forces/Faiblesses ;
Externe = Opportunités/Menaces
Toujours séparer ce qui appartient à l’entreprise de ce
qui vient du marché.
Structure = Division + Coordination +
Autorité
Code simple pour expliquer pourquoi une structure
existe.
Stratégie ≠ Tactique
Stratégie = orientation ; Tactique =
actions/moyens
Très utile pour les questions de définition.
3. Tableau final des questions d’examen probables
Question Plan de réponse rapide
Définir la gestion. Définition + PODC + ressources/objectifs + performance.
Comparer gestion et
management.
Gestion = techniques et efficacité ; management = conduite humaine et
collective.
Analyser une entreprise. Finalités + facteurs de production + taille/secteur/juridique + fonctions.
Identifier une structure. Regroupement par fonction/produit/zone/projet + avantages/limites.
Analyser l’environnement. Micro/macro + PESTEL + SWOT + turbulence/complexité/hostilité.
Proposer une stratégie. Vision/mission/objectifs + SWOT + type de stratégie + tactiques.
